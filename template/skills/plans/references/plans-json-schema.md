## Field Sources

| Field | Source |
|---|---|
| `file` | filename |
| `title` | H1 heading in the plan file |
| `status` | frontmatter |
| `priority` | frontmatter |
| `owner` | frontmatter |
| `type` | frontmatter |
| `depends_on` | frontmatter |
| `blocks` | frontmatter |
| `last_updated` | frontmatter |
| `start_date` | explicit field in frontmatter or earliest phase date |
| `overall` | `## Status` banner Overall line |
| `eta` | banner or explicit frontmatter field |
| `in_flight` | explicit frontmatter field (`true` = actively being worked now) |
| `next_action` | `## Status` banner Next action line |
| `phase_summary` | per-phase lines in `## Status` banner |
