This page reads plans.json over HTTP, so it needs a local server. Run one of these from your project root:

python -m http.server 8080  (Python 3)
npx serve -l 8080  (Node.js)
php -S localhost:8080  (PHP)

Then open http://localhost:8080/plans/roadmap.html

About this view
This dashboard renders plans.json into a Gantt timeline, a blocking-chain dependency graph, and a filterable plan list. Run /plans sync to regenerate the data.
